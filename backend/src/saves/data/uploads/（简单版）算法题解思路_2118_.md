/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var deleteDuplicates = function(head) {
    // current作为要操作的指针控制新列表数据的变化
    //还需要一个指针进行查找重复值
    let current = head;
    while(current){
        let repeate = current.next;
        while(repeate&&repeate.val===current.val){
            repeate = repeate.next;
            current.next = repeate
        }
        current = current.next
    }
    return head
};

## 110，平衡二叉树

1. **检查子树是否平衡**:
   - 如果左子树或右子树不平衡（即返回 `-1`），则整棵树不平衡，函数返回 `-1`。
2. **检查高度差**:
   - 如果左右子树的高度差超过 `1`，则整棵树不平衡，函数返回 `-1`。

1. **返回树的高度**:
   - 如果当前节点是平衡的，函数返回当前节点的高度，即 `Math.max(left, right) + 1`。
2. **检查整棵树是否平衡**:
   - `isBalanced` 函数调用 `depth(root)` 并检查其返回值。如果返回值不是 `-1`，则表示树是平衡的，返回 `true`；否则返回 `false`。

## 111.二叉树的深度

相较于最大深度，需要判断是否为空，为空就小

**区别**：

- 最大深度关注的是树的高度，而最小深度关注的是从根到最近叶子节点的最短路径。
- 最大深度总是大于或等于最小深度，因为最小深度是树的最短路径，而最大深度是树的最长路径。

**联系**：

- 最大深度和最小深度都是基于树的深度概念，它们都是用来描述树结构的特性。
- 它们都可以通过递归算法来计算，算法的基本思想是遍历树中的每个节点，并计算从该节点到叶子节点的路径长度。
